# @ryziz-shopify/ryziz

Template for creating Shopify embedded apps on Firebase with file-based routing.

## Quick Start

```bash
npx @ryziz-shopify/ryziz init
```

This will:
- Scaffold a new Shopify app project
- Install all dependencies
- Set up file-based routing
- Configure Firebase and Shopify integration

## Next Steps

After initialization:

```bash
# Link to your Shopify app
npm run link

# Start development server
npm run dev
```

## What's Included

- File-based routing (`page.*`, `app.*`, `api.*`, `webhooks.*`)
- Shopify OAuth and App Bridge
- Firebase Cloud Functions + Hosting + Firestore
- Hot reload development environment
- Zero configuration
- Best practices from 10 years of production experience
